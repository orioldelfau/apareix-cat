# Decisions Pendents del Founder

Aquestes decisions serveixen per convertir Apareix en una oferta real i prioritzar el primer MVP.

## 1. Mercat Inicial

Preguntes:

- A quina zona vols comencar: Catalunya sencera, Barcelona, Girona, Maresme, Valles, Costa Brava o una ciutat concreta?
- Vols vendre en catala, castella o bilingue?
- Prefereixes restaurants premium, restaurants de menu, cafeteries, bars, take away o qualsevol restaurant independent?
- Tens ja restaurants coneguts als quals puguis vendre els primers pilots?

Recomanacio inicial:

Comencar amb una zona concreta i restaurants independents amb ticket mitja suficient per pagar 150-300 EUR/mes.

## 2. Oferta Inicial

Preguntes:

- Vols posicionar-ho com a servei fet per tu o com a software automatitzat?
- Vols cobrar setup des del primer client?
- Quin preu mensual et sentiries comode defensant?
- Vols fer permanencia minima de 3 mesos?

Recomanacio inicial:

Vendre com a servei recurrent automatitzat, no com a SaaS pur. El client compra tranquil.litat i resultat, no acces a una eina.

## 3. Operacio

Preguntes:

- Vols que Apareix respongui ressenyes directament o nomes prepari respostes?
- Vols que el client aprovi posts abans de publicar?
- Quantes hores setmanals pots dedicar als primers 5 clients?
- Vols fer tu les vendes inicials per telefon, email, porta freda o LinkedIn?

Recomanacio inicial:

Durant els primers clients, aprovacio obligatoria per ressenyes negatives i canvis sensibles. Posts positius i informatius poden anar amb aprovacio simple.

## 4. Tecnologia

Preguntes:

- Vols que construim una eina interna ara dins d'aquest repo?
- Tens preferencia per Next.js, Supabase, Vercel o un altre stack?
- Tens compte Google Cloud preparat per demanar acces a Business Profile API?
- Tens OpenAI API key o vols que deixem la integracio preparada pero sense clau?

Recomanacio inicial:

Construir primer un panell intern amb Next.js + Supabase. Integrar Google quan l'oferta i els primers clients estiguin validats.

## 5. Marca

Preguntes:

- El nom definitiu es Apareix?
- Vols domini principal `apareix.cat`?
- Vols una marca propera i local o mes tecnològica i premium?
- Tens logo, colors o preferencies visuals?

Recomanacio inicial:

Mantenir `Apareix`: es curt, local, memorable i connecta directament amb el benefici.

## 6. Primeres Accions

La sequencia recomanada:

1. Definir oferta i preu.
2. Crear landing simple per restaurants.
3. Crear formulari d'auditoria gratuita.
4. Preparar plantilla d'auditoria manual.
5. Contactar 20 restaurants.
6. Tancar 3 pilots pagats.
7. Construir panell intern per operar-los millor.

## Respostes Minimes que Necessito Ara

Per decidir el primer sprint, necessito aquestes respostes:

1. Zona inicial on vols vendre.
2. Tipus de restaurant ideal.
3. Preu mensual que vols provar.
4. Si vols setup inicial o no.
5. Si vols construir ja la web publica d'Apareix o primer l'eina interna.
6. Stack preferit, si en tens.
